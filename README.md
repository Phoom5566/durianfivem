# DURIAN Surname Store

## วิธีเอาขึ้นเว็บจริง (ฟรี ใช้เวลา ~10 นาที)

### วิธีที่ง่ายที่สุด — Vercel
1. สมัครบัญชีฟรีที่ https://vercel.com (ล็อกอินด้วย GitHub ได้เลย)
2. อัปโหลดโฟลเดอร์นี้ขึ้น GitHub เป็น repo ใหม่ (หรือใช้ Vercel CLI ก็ได้ ดูด้านล่าง)
3. ใน Vercel กด "Add New Project" แล้วเลือก repo นี้ → กด Deploy
4. รอ ~1 นาที จะได้ลิงก์เว็บจริง เช่น `durian-store.vercel.app`

### หรือใช้คำสั่งจากเครื่องตัวเอง (ต้องลง Node.js ก่อน)
```
npm install
npm run build
npx vercel --prod
```

### ทดสอบก่อนขึ้นจริง (รันในเครื่อง)
```
npm install
npm run dev
```
จะเปิดที่ http://localhost:5173

## ⚠️ สำคัญ: เรื่องข้อมูลออเดอร์/รายชื่อเจ้าของ

โค้ดนี้เดิมรันอยู่ใน Claude artifact ซึ่งมีระบบเก็บข้อมูลถาวรแบบแชร์ให้ทุกคนเห็นร่วมกันในตัว
แต่เมื่อเอาไป deploy เป็นเว็บจริงข้างนอก ระบบนั้นจะไม่มีอยู่ — ผมทำ fallback ให้เก็บด้วย
`localStorage` แทน ซึ่ง**เก็บได้เฉพาะในเบราว์เซอร์เครื่องนั้นๆ ไม่แชร์ข้ามคนซื้อ**
(ลูกค้าคนหนึ่งซื้อ อีกคนจะไม่เห็นออเดอร์นั้น และหน้า "เจ้าของ" จะไม่อัปเดตให้ทุกคนเห็นตรงกัน)

ถ้าต้องการให้ข้อมูลออเดอร์จริงเก็บส่วนกลางแบบใช้งานร้านค้าจริงได้ ต้องต่อฐานข้อมูลจริง เช่น:
- **Supabase** (ฟรี, ตั้งค่าง่าย, มี database + API ให้พร้อม) — แนะนำ
- **Firebase Firestore** (ฟรี, ของ Google)
- เขียน backend เอง (Node.js + database)

บอกได้เลยถ้าอยากให้ผมต่อให้เว็บนี้เชื่อมกับ Supabase หรือ Firebase เพื่อให้ข้อมูลออเดอร์เก็บจริง
แชร์เห็นตรงกันทุกคนเมื่อขึ้นเว็บจริงแล้ว
